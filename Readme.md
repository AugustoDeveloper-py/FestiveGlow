## Customization ğŸ¨

### Colors

The animation uses Material Design colors. Main colors can be modified in the CSS:

```css
:root {
  --primary-red: #e53935;
  --primary-blue: #1a237e;
  --accent-gold: #ffd700;
}
```

### Animation Timing

Adjust animation speeds by modifying the keyframe durations:

```css
.santa {
  animation: float 3s ease-in-out infinite;
}
```
